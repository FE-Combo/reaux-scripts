## reaux-scripts
Reaux 的配置和脚本

### 安装
```bash
npm i reaux-scripts -g
```

### 使用
```bash
reaux-scripts dev ## 应用启动
reaux-scripts build ## 应用打包
reaux-scripts eject ## 暴露配置
```

### 环境变量
ASSET_PREFIX: 静态资源前缀
PORT: 端口号
